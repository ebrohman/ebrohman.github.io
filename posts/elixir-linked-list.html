Reversing a linked list using recursion


Working with lists in Elixir is pretty fun and intuitive. We can use pattern matching and recursion to destructure our list and reassemble a new one, because the starting list is immutible.

In order to accomplish this we define a rev function within our LinkedList module with multiple signatures to pattern match against our list and eventually bottom out when we encounter an empty list.

Our first and only public rev function takes a default argument to pass to our private _rev function. We then begin to fill this empty list in reverse order. 

            
              defmodule LinkedList do
                def rev(list, reversed \\ []) do
                  _rev(list, reversed)
                end

                defp _rev([], reversed) do
                  reversed
                end

                defp _rev([head | tail], reversed) do
                  _rev(tail, [head | reversed])
                end
              end
